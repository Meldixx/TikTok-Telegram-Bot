# TikTok Telegram Bot

Телеграм-бот, который скачивает видео из TikTok по ссылке и отправляет их в максимальном доступном качестве.

## Требования
- Node.js 18+
- Токен Telegram бота (получите у [BotFather](https://t.me/BotFather))

## Установка
1. Склонируйте репозиторий и перейдите в папку проекта.
2. Скопируйте файл окружения и укажите токен бота:
   ```bash
   cp .env.example .env
   # Откройте .env и вставьте токен в BOT_TOKEN
   ```
3. Установите зависимости:
   ```bash
   npm install
   ```

## Запуск
```bash
npm start
```
Бот автоматически запустится и начнет принимать сообщения.

## Использование
Отправьте боту любую ссылку на видео TikTok. Он скачает ролик через публичное API `tikwm.com` и вернет файл с подписью.

## Примечания
- Ссылки обрабатываются с помощью регулярного выражения `tiktok.com`.
- Для предотвращения спама бот отправляет статус загрузки перед отправкой видео.
- Если API недоступно или ссылка некорректна, бот уведомит вас об ошибке.
